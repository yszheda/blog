# Galoisplusplus Blog - Now Powered by Pelican

This blog has been migrated from Octopress to Pelican static site generator.

## Quick Start

### Preview the Site
```bash
make serve
```
Then visit http://localhost:8000

### Build for Production
```bash
make publish
```

## Content

The blog contains 84 articles covering topics such as:
- Computer science and technology
- Cocos2d-x game development
- CUDA and GPU programming
- LaTeX and documentation
- Classical music reviews

## Technical Details

### Build System
- **Static Site Generator**: Pelican 4.11.0
- **Language**: Python 3.14
- **Theme**: pelican-bootstrap3
- **Syntax Highlighting**: Pygments (monokai style)
- **LaTeX Support**: MathJax 2.7.9

### Original Blog
This blog was previously built with Octopress and Jekyll. The migration to Pelican was completed to simplify the build process and improve performance.
